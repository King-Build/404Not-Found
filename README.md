# 404 Not Found Page

## 🔎 Overview  
This is a custom **404 Not Found** page created using pure **HTML** and **CSS**.  
It displays a clean and user-friendly message when a user tries to access a non-existent page.

## 🛠 Technologies Used

- CSS3
